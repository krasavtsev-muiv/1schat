# Веб-сервис чата с интеграцией 1С

Веб-сервис для организации коммуникаций между сотрудниками и клиентами через веб-чат с интеграцией в 1С:Предприятие.

## Технологический стек

- **Frontend**: Next.js (React)
- **Backend**: Node.js + Express
- **База данных**: PostgreSQL
- **WebSocket**: Socket.io для реального времени
- **Интеграция**: REST API для синхронизации с 1С

## Структура проекта

```
app/
├── backend/          # Express API сервер
├── frontend/         # Next.js приложение
├── database/         # SQL схемы и миграции
└── docs/            # Документация
```

## Установка и запуск

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
